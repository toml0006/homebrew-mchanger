# homebrew-mchanger

Homebrew tap for [mchanger](https://github.com/toml0006/mchanger), a CLI tool to control SCSI media changer devices on macOS.

## Installation

```sh
brew tap toml0006/mchanger
brew install mchanger
```

Or directly:

```sh
brew install toml0006/mchanger/mchanger
```
